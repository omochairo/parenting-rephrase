# 育児の言い換え - 気づきの言葉がけ

育児の様々なシーンで使える「ポジティブな言い換え」を1000個集めたWebアプリケーションです。
押しつけがましくなく、気づきを与えることを目的としています。

## 特徴

- 🔍 **リアルタイム検索**: キーワードで瞬時に言い換えを検索
- 📂 **カテゴリー分類**: 「叱る場面」「褒める場面」など状況別にフィルター
- ❤️ **お気に入り機能**: 気になる言い換えを保存（ブラウザに保存されます）
- 📱 **スマホファースト**: モバイルでも見やすいカードデザイン
- 🚀 **完全動作**: サーバー不要で動作する静的アプリケーション

## 技術スタック

- React
- TypeScript
- Vite
- Vanilla CSS (変数ベースのデザインシステム)

## セットアップ手順

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

## ビルドとデプロイ

このプロジェクトは静的サイトとしてデプロイ可能です。

### ビルドコマンド

```bash
npm run build
```

ビルドが完了すると `dist` ディレクトリにファイルが生成されます。

### デプロイ方法

#### Netlify / Vercel / Cloudflare Pages

1. リポジトリをGitHubにプッシュ
2. 各ホスティングサービスと連携
3. 以下の設定でデプロイ:
   - Build command: `npm run build`
   - Publish directory: `dist`

#### GitHub Pages

プロジェクトの設定によっては `vite.config.ts` の `base` 設定が必要になる場合があります。
現在の設定はルート (`/`) での公開を想定しています。

## ディレクトリ構成

- `src/components`: UIコンポーネント
- `src/data.ts`: 言い換えデータ（1000個）
- `src/types.ts`: TypeScript型定義
- `src/index.css`: グローバルスタイルとデザインシステム
- `src/App.css`: アプリケーション固有のスタイル
